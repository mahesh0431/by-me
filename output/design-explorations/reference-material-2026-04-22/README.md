# Reference Material Design Explorations

This folder contains page-specific design explorations based on the attached reference image from the thread.

Design anchor:
- soft Material-like minimalism
- warm off-white surfaces
- Roboto-like sans feel
- restrained teal accents
- spacious, writing-first layout

Page sets:
- `about/`
- `blog-index/`
- `blog-post/`
- `archive/`
- `post-detail/` (bonus alternate article-detail set)

Each page folder contains:
- 3 PNG variations
- `prompts.md` with a short description for each variation
